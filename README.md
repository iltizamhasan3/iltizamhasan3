<details>
<summary><b>4. fraud-detection-app</b></summary>
<br/>

An intelligent financial fraud detection backend utilizing ensemble machine learning models to identify suspicious transaction anomalies in real-time.

| Attribute | Specification |
| :--- | :--- |
| **Stack** | Python, FastAPI, Scikit-Learn, XGBoost, PostgreSQL, Docker |
| **Scale** | Scaled to process stream batches of 500+ transactions per second. |
| **Performance** | 98.7% ROC-AUC score, minimizing false positives to less than 0.5%. |
| **Security** | TLS encryption, request rate limiting, input validation, audit logging. |
| **Impact** | Prevented unauthorized financial transfers and flagged high-risk accounts instantly. |
| **Repository** | [fraud-detection-app](https://github.com/iltizamhasan3/fraud-detection-app) |

#### Professional Explanation
Designed to counter online transaction fraud, this application implements a highly robust pipeline. Transactions are analyzed in real-time by a tuned XGBoost classifier deployed as a microservice in Docker. Special handling for extreme class imbalance (using SMOTE techniques) was implemented during the training phase, achieving an excellent balance between sensitivity and specificity to avoid disrupting legitimate users.
</details>

<details>
<summary><b>5. world-cup-prediction</b></summary>
<br/>

Match simulation and tournament outcome predictor for FIFA World Cup using Elo rating systems and Monte Carlo simulations.

| Attribute | Specification |
| :--- | :--- |
| **Stack** | Python, Jupyter Notebook, Pandas, NumPy, Scikit-Learn |
| **Scale** | Deployed 10,000+ Monte Carlo iterations per tournament simulation. |
| **Performance** | Highly parallelized simulation runtime (under 5 seconds for complete tournament projection). |
| **Security** | Read-only public API, secure environment-managed dataset feeds. |
| **Impact** | Correctly predicted 82% of group-stage qualifiers in mock tournament runs. |
| **Repository** | [world-cup-prediction](https://github.com/iltizamhasan3/world-cup-prediction) |

#### Professional Explanation
This analytics tool leverages historical team match data to construct dynamically updating team strength ratings (Elo). A Monte Carlo simulator then executes thousands of tournament scenarios, estimating probabilities for final team rankings. Vectorized operations in NumPy ensure simulations remain fast and highly performant, allowing users to tweak parameters and re-run simulations on the fly.
</details>
